# Try to Lucky

How to Use :

- to play the game, run Lucky_Game file
- follow and input as instructured in the program
- have fun and enjoy the game

The Rules :

What's Your Digit
- Guess 4 Digit number
- True Numer is Random 
- Number from 0000 to 9999
- this game can be played with multiple players

Coin Guess
- this game based on coins money
- guess between "angka" or "gambar"
- this game can only be played by one player
